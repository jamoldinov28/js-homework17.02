JavaScript Map Metodlari

Map obyekti kalit-qiymat juftliklarini saqlash uchun ishlatiladi. Bu bo'limda siz Map metodlari bilan tanishasiz.

set() Metodi

set() metodi Map ga yangi kalit-qiymat juftligini qo'shadi:

const map = new Map();
map.set('ism', 'John');
map.set('kasb', 'Oshpaz');
console.log(map);

get() Metodi

get() metodi kalit bo'yicha qiymatni qaytaradi:

const ism = map.get('ism');
console.log(ism); // 'John'

has() Metodi

has() metodi Map da kalit mavjudligini tekshiradi:

console.log(map.has('kasb')); // true
console.log(map.has('yosh')); // false

delete() Metodi

delete() metodi kalit bo'yicha qiymatni o'chiradi:

map.delete('kasb');
console.log(map);

clear() Metodi

clear() metodi Map ni tozalaydi:

map.clear();
console.log(map); // Empty Map

size Xususiyati

size Map dagi elementlar sonini qaytaradi:

map.set('ism', 'Ali');
console.log(map.size); // 1
